Modelo de Cajas en CSS

¿Qué es el modelo de cajas?

El modelo de cajas es la forma en que CSS representa y calcula el espacio que ocupa cada elemento en una página web. Todo elemento es una caja compuesta por: contenido, padding, borde y margen.

Entender cómo funciona te permite diseñar interfaces bien estructuradas y responsivas.
